Marko Widgets: UI Components Playground w/ Webpack
==================================

## Getting Started

```bash
npm install
npm run dev
```

## Production Build
```bash
npm run build
npm start
```

## Fetures

- Support a timer count down by inputing a future time.
- If the screen width less then 760px, the page will show like picture 2.
